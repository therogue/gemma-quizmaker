## One-liner
AI assistant w/ quiz generator to help learning various topics.

## Goals
Allow user to:
- Ask a question about a topic
- Get high-level overview of topic
- Get a short quiz to reinforce information retention
- Continue diving into more details in the same conversation
- Have multiple conversations for various topics
- Get periodic questions from old topic quizzes in the same conversation (spaced repetition)
- Get periodic questions from old quizzes when the questions have been answered incorrectly
- Upload own materials (files, images, etc) to have app parse them and present information in overview / quiz
- Receive recommendations for follow-up topics

## Deliverables
- Single offline user app
- Backend with database for storing conversations
- Simple chat UI with conversation picker
- Web search tools
- File read tools
- Graph / equation generator tools

## Competition Guideline
Future of Education: Reimagine the learning journey by building multi-tool agents that adapt to the individual and empower the educator through seamless integration.
